# API - Controle de Atendimento

## Sobre
API REST para gerenciamento de atendimentos da clínica da UCSAL.  
Responsável por regras de negócio e persistência de dados, sendo consumida pelo front-end.

---

## Tecnologias
- Java + Spring Boot
- Spring Data JPA
- Spring Security
- PostgreSQL

---

## Arquitetura
Controller → Service → Repository → Database
